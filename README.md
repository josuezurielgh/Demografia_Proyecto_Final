# 🌎 Proyecto Demografia 🗺️
El trabajo que se presenta a continuación constituye el proyecto final del curso de Demografía,  enfocandose en el análisis del estado de Chihuahua a nivel poblacional. 

Tal y como señaló el sociólogo y demógrafo Rupert B. Vance:

_"la demografía estudia a la población humana recurriendo tanto a las ciencias sociales como a las ciencias exactas. Su fuerza radica en que traduce la vida y el destino humano al lenguaje exacto de los números y las fórmulas"_ .

Bajo esta premisa buscamos dar una razón a los datos mediante el apoyo de herramientas como Excel y R, desarrollando asi, un análisis completo, es decir cuantitativo que permite comprender de mejor manera el comportamiento de la mortalidad en la región, y cualitativo, el otro lado más alla de las cifras que complementa nuestra investigación.

## 🖇️ Objetivos del proyecto
 🟡 Construcción de tablas de mortalidad
 
 - Limpieza de bases de datos para la obtención correcta de datos
 - Implementación de código en R para fórmulas demográficas
 - Manipulación de herramientas como Excel para realizar cálculo prevíos
 - Análisis de datos obtenidos

## 🖥️🧠 Material de apoyo
Se hizo uso de:
- 📊 Excel
- 🗺️ INEGI (_base de datos_)
- 📦 R / RStudio (_entorno de programación y ejecución_)
- 📄 Quarto (_documentos dinámicos y visualización_)
